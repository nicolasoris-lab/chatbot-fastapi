# Chatbot
Chatbot con FastAPI.

# Entorno virtual
## Crearlo
```bash
py -3 -m venv .venv
```

## Activarlo
```bash
.venv\Scripts\activate
```

## Desactivarlo
```bash
deactivate
```

# Instalar Dependencias
```bash
pip install -r requirements.txt
```

# Variables de Entorno
Crear archivo `.env` en base al archivo `.env.template` y agregarlas las variables necesarias.
